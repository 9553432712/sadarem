Calculation Guidelines: -

1. Calculation will be done separately on left side and right side parts.
2. If any side the value is >= 90, then no calculation will be done. That value will be directly taken as temporary result.
3. If there is no value >=90, then the actual calculation part will be done and the final value will be taken as temporary result.
4. Calculation will be done on the following principles.
a) First sort all the values in descending order.
b) Select the first one as value a, and the second one as value b.
c) Supply these a,b values to the following formula.
a+((b*(90-a))/90);
d) The above formula gives a result. And this result will be taken as a value for the next calculation.
e) In the next calculation the b value will be the next value in the descending order.
f) This procedure will be repeated for respective number of times. Finally we will get a temporary result of one side.
g) Follow the same procedure for the other side. Then the we will get the temporary result of other side.
h) Now compare the both temporary results. The maximum between them will be taken as a value and the other is taken as b value.
g) Atlast apply these two values to the formula given in point[c].
h) The output of the formula will be taken as final result and will be entered into the database.
